# Опционально:
insmod kqemu.ko

##############################

QEMU предоставляет специальную команду для создания жесткого диска,

которая называется qemu-img

Создание образа:
ТИПЫ: Лучший (для qemu) из них называется qcow (или qemu copy-on-write). 


1) создаем образ диска на 128 МБ:
qemu-img create disk.img 128M

2) Теперь у нас есть эмулируемый диск (disk.img):
Следующим шагом будет инсталляция системы на жесткий диск
qemu -hda disk.img -cdrom ./ИНСТАЛЯЦИОННЫЙ_ДИСК -boot d
qemu-system-x86_64 -enable-kvm -hda ./ДИСК -cdrom ./ИНСТАЛЛЯЦИОННЫЙ_ДИСК -boot d -m 1024

>> образ жесткого диска задается с помощью опции hda, а компакт-диск
(файл, где располагается образ) - с помощью опции cdrom. Опция boot
позволяет загрузиться с CD-ROM. Аргумент d указывает загружаться с
CD-ROM, a - с флоппи-диска, c указывает на загрузку с жесткого диска
(по умолчанию), а n - загрузку с сети.

3) Установка требует перезагрузки. В этом месте можно закончить эмуляцию
(Ctrl-C в окне qemu).

4) Теперь можно загрузить свежеустановленную
операционную систему с помощью следующей команды:
qemu -hda disk.img
qemu-system-x86_64 -enable-kvm -hda ./ДИСК -m 1024

>> Эта команда просто эмулирует стандартный PC (опция по умолчанию) с
жестким диском, представленным файлом образа disk.img
