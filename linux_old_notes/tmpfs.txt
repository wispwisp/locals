Предлагаю, использовать несколько каталогов с помощью tmpfs, а именно:
/tmp - Многие программы помещают туда временные файлы и поместив эту
директорию в память, мы увеличим время отклика программы.
/home/$username$/.cache - каталог с временными файлами пользователя

Просто допишем в /etc/fstab строчки:
tmpfs /tmp tmpfs defaults 0 0
tmpfs /home/$username$/.cache tmpfs size=200M 0 0

///////////////////////////////////////////////////////////////////////////////

// + :
mount tmpfs -t tmpfs /tmp
включая кэш браузера
mount tmpfs -t tmpfs /home/$username$/.cache/chromium	# а лучше весь кэш
mount tmpfs -t tmpfs /var/tmp/ИМЯЮЗЕРА-megabaks
и прочее...

если syslogd будет дёргать винт - добавить минус в начало каждого пути
к журналу в файле /etc/syslog.conf.


#testing in Ubuntu
tmpfs   /tmp    tmpfs   defaults        0       0
tmpfs   /home/$username$/.cache tmpfs   size=400M       0       0
tmpfs   /var/tmp        tmpfs   defaults        0       0
